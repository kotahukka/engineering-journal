# AI Reviewers

This repo uses AI reviewers to provide early feedback on PRs.

## What they review

- CI and workflow YAML
- Config files (YAML/JSON)
- Scripts (shell/Python)

## What they don’t review

- Journal entry Markdown (most docs content)

## What they focus on

- Correctness and breakage risk
- GitHub Actions safety
- Security issues and secrets exposure

## Note

AI reviewers do not approve PRs. Human review is still required.
